# Networking for Cybersecurity

This folder contains notes and hands-on observations related to networking concepts from a cybersecurity and SOC analyst perspective.

## Topics Covered
- Network communication flow (client to server)
- OSI vs TCP/IP models and security relevance
- TCP vs UDP behavior
- Ports, protocols, and exposed services
- DNS basics and abuse scenarios
- HTTP vs HTTPS and encryption fundamentals

## Focus
- How attackers use network protocols
- How defenders detect abnormal traffic
- Mapping attacks to network layers

All notes are based on practical learning and traffic analysis using tools like Wireshark.

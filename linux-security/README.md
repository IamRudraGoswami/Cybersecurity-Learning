# Linux Security Fundamentals

This folder documents Linux operating system concepts relevant to cybersecurity and SOC operations.

## Topics Covered
- Linux filesystem structure
- Users, groups, and permissions
- Authentication and authorization
- Log analysis (auth.log, syslog)
- Common Linux misconfigurations

## Security Focus
- Privilege escalation paths
- Brute-force attack indicators
- Hardening Linux systems
- Detecting suspicious activity via logs
